## 目录说明

本目录包含游戏所有 UI 文本的多语言翻译数据。格式基于 **I2 Localization** 插件。

**支持语言：**
English / Russian / German / French / Spanish / Portuguese (Brazil) / Chinese (Simplified) / Japanese / Korean / Thai / Italian / Turkish / Chinese (Traditional) / Polish / Indonesian

**数据加载机制：** 与 crc/ 目录相同，通过 `LOC_CRC.json` 管理校验，按需从服务器更新本地化文本。

## I2Localization 数据格式

```
[i2category]<分类名>[/i2category]
Keys[*]Type[*]Description[*]English[*]Russian[*]German[*]...[语言列]
<key>[*][*]<备注>[*]<EN文本>[*]<RU文本>[*]<DE文本>[*]...
[ln]
<key2>[*][*]<备注>[*]<EN文本>[*]...
```

- `[*]` = 列分隔符
- `[ln]` = 行分隔符  
- `[i2category]...[/i2category]` = 分类标签

**语言列顺序：**
English → Russian → German → French → Spanish → Portuguese (Brazil) → Chinese (Simplified) → Japanese → Korean → Thai → Italian (Italy) → Turkish → Chinese (Traditional) → Polish → Indonesian
