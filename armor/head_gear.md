# 头部护具（Helmets & Head Gear）属性数据库

> Modern Ops v9.79 | 数据来源：CRCD28.json (wt=2) + CRCD33.json (升级路径)

> 部分头部装备仅为外观装饰（无属性值），以 ☆纯装饰 标注。
> 提取日期：2026-05-09

## 字段说明

| 字段 | 含义 |
|------|------|
| stArm | 护甲值 |
| stAHe | 额外生命值 |
| stMSp | 移动速度加成 % |
| stPHe | 爆头伤害减免 % |
| stPEx | 爆炸伤害减免 % |
| stAGr | 额外携带手雷数 |
| stDGr | 手雷伤害加成 % |
| stDHe | 爆头伤害加成 % |
| stD{XX} | 受到对应武器类型伤害减免 % （01=刀,03=手枪,04=步枪,06=轻机枪,07=霰弹,10=狙击,17=冲锋枪）|
| stPD{XX} | 受到对应武器类型穿甲伤害减免 % |
