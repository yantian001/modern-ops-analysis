## 字段说明

| 字段代码 | 含义 |
|---------|------|
| stArm | 护甲值（Armor），叠加到角色最大护甲 |
| stAHe | 额外生命值（Health），叠加到角色最大血量 |
| stMSp | 移动速度加成 % |
| stPEx | 爆炸伤害减免 % |
| stDGr | 手雷伤害加成 % |
| stPD04 | 受到突击步枪穿甲伤害减免 % |
| stPD06 | 受到轻机枪穿甲伤害减免 % |
| stPD07 | 受到霰弹枪穿甲伤害减免 % |
| stPD10 | 受到狙击枪穿甲伤害减免 % |
| stPD11 | 受到特殊武器穿甲伤害减免 % |
| stPD17 | 受到冲锋枪穿甲伤害减免 % |
| ena | 1=基础版(可购买), 2=升级版(由前一级升级获得), 4=特殊限定 |
| sc | 购买费用（tPv=金币, tPb=蓝图, tPr=钻石） |
| 升级费用 | CRCD33.json中 sc 字段（升级到下一级的费用） |
