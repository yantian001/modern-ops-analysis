## 数据结构说明

### CRCD4.json 字段

| 字段 | 含义 |
|------|------|
| \i\ | 技能ID |
| \l\ | 技能等级（1-5） |
| \	\ | 技能类型（1=攻击/2=强化/3=防御/4=生命/5=精准） |
| \o\ | 技能子类型ID（对应技能效果的具体分类） |
| \
l\ | 解锁所需玩家等级 |
| \en\ | 是否启用（1=启用） |
| \\ | 技能效果 JSON 数组（见下方） |
| \sc.tPg\ | 升级所需技能点（Prestige Points） |
| \sc.sc_id\ | 赛季挑战ID（关联赛季任务） |

### v 字段（技能效果数组）格式

```json
[{"t": 1, "v": "10", "vt": 2}]
```

| 字段 | 含义 |
|------|------|
| \	\ | 效果类型ID（对应游戏内具体属性，如武器伤害、移速等） |
| \\ | 效果数值 |
| \t\ | 数值类型（1=绝对值，2=百分比） |

### LOC_CRCD1.json 本地化键名规则

| 键名格式 | 含义 |
|---------|------|
| \_{id}_name\ | 技能名称 |
| \_{id}_desc\ | 技能描述（含 \{[value]}\ 占位符） |
| \_{id}_subType\ | 技能子类型名称 |
| \_{id}_{lv}_val\ | 第lv级的效果数值文本（部分技能有） |
| \_type_{n}\ | 技能类型名称（n=1-5） |
