## 武器属性字段说明

### CRCD27.json 字段映射

| JSON字段 | 含义 | 说明 |
|---------|------|------|
| `i` | 武器ID | 唯一标识符（升级版ID通常>1000） |
| `sn` | 系统名称 | 内部名称，对应资源文件名 |
| `wt` | 武器类型 | WeaponType枚举值 |
| `stDa` | 伤害 (Damage) | 单次命中基础伤害值 |
| `stRa` | 射速 (FireRate) | 每分钟射弹数（RPM） |
| `stDi` | 射程 (Range) | 有效射程评分 |
| `stDe` | 精度 (Accuracy) | 精准度评分 |
| `stRt` | 换弹速度 (Reload) | 换弹速率系数 |
| `stSp` | 移速 (MoveSpeed) | 携弹移速系数 |
| `stPo` | 综合威力 (Power) | 游戏内武器评分 |
| `am` | 弹匣容量 | 单弹匣装弹发数 |
| `amt` | 备弹总量 | 含弹匣的总弹药量 |
| `nl` | 解锁等级 | 所需玩家等级 |
| `rar` | 稀有度 | 1=普通 2=稀有 3=史诗 |
| `kr` | 暴击率 | 暴击伤害加成 |
| `ena` | 启用状态 | 1=基础版 2=升级版 |
| `bS1~bS6` | 蓝图槽配件ID | 预装配件的ID |
