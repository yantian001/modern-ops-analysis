## 武器属性字段说明

### 本地配置属性（.asset 文件中）

| 属性 | 说明 |
|------|------|
| 系统名称 | 武器的唯一标识符，格式：前缀_型号名 |
| 武器类型 | WeaponType 枚举（共18种） |
| 模型缩放倍率 | 菜单/UI中的模型显示缩放 |
| 配件槽位 | 支持的配件类型（OPTIC/MUZZLE/UNDERBARREL/MAGAZINE/BUTT/SKIN/TRINKET/AMMO） |
| aimClampedSpeed | 瞄准时最大移动速度限制 |
| RecoilDampForce | 后坐力阻尼强度 |
| RecoilSpeed | 后坐力抬升速度 |

### 服务器下发属性（Photon 网络传输）

| 属性 | Key | 说明 |
|------|-----|------|
| 弹速 | Velocity(97) | 子弹飞行速度 |
| 射速 | Rapidity(94) | 开火间隔毫秒 |
| 换弹时间 | ReloadTime(93) | 毫秒 |
| 弹匣容量 | Ammo(92) | 发数 |
| 备弹总量 | AmmoTotal(91) | 发数 |
| 近程 | ShortRange(74) | 米 |
| 中程 | MediumRange(73) | 米 |
| 远程 | LongRange(72) | 米 |
| 站立散布 | MinStand(59)~IncreaseStand(56) | 精度值 |
| 蹲伏散布 | MinCrouch(63)~IncreaseCrouch(60) | 精度值 |
| ADS散布 | MinStandOptics(67)~IncreaseStandOptics(64) | 精度值 |
| 出枪时间 | ShowingDurationMSec(45) | 毫秒 |
| 收枪时间 | HidingDurationMSec(44) | 毫秒 |
| 开镜时间 | ZoomInDurationMSec(43) | 毫秒 |
| 拉栓 | BoltAction(103) | 布尔值 |
| 换弹类型 | ReloadingType(42) | 0=整匣, 1=逐颗 |
