## 2. 武器数据层

### BattleWeapon — 运行时武器数据对象

继承自 `BaseWeapon`，是战斗中每把武器的运行时表示。

```csharp
namespace Battle.CombatControl
{
    public abstract class BattleWeapon : BaseWeapon
    {
        // ── 状态 ──────────────────────────────────────────
        public WeaponState PrevState    { get; }  // 上一个状态
        public WeaponState CurrentState { get; }  // 当前状态

        // ── 视图引用 ──────────────────────────────────────
        public BaseWeaponViewController WeaponView { get; protected set; }

        // ── 模型名 ────────────────────────────────────────
        public string ModelName { get; private set; }        // prefab名，用于加载

        // ── 武器信息（来自prefab组件） ────────────────────
        public WeaponSettings WeaponInfo { get; private set; }

        // ── 武器配件 ──────────────────────────────────────
        public BattleWeaponParts WeaponParts { get; private set; }

        // ── 其他 ──────────────────────────────────────────
        public int  ShotCount  { get; }           // 已发射子弹数（全自动计数）
        public bool CanDrop    { get; }           // 是否可以被丢弃
        public int  TimeWhenReloadingWillBeFinished { get; }  // 换弹结束时间戳（毫秒）

        // ── 静态工厂方法 ──────────────────────────────────
        // 根据武器类型决定创建哪种 BattleWeapon 子类
        public static BattleWeapon CreateBattleWeapon(
            byte slotIndex, WeaponType type, string systemName,
            KitWeaponsDC.WeaponDC wpnData, IBattlePlayer owner, ...);

        // KillStreak武器专用（Juggernaut/Flamer等）
        public static BattleWeapon CreateKillstreakBattleWeapon(...);

        // ── 核心操作 ──────────────────────────────────────
        public Shot  Fire(PlayerMotionState motionState, Transform shotOrigin, out Ray ray);
        public void  StartReloading();
        public bool  InterruptReloading(short loadedAmmo, short reservedAmmo);
        public void  Bolt(Action OnComplete);
        public void  SwitchToThis(int timeWhenSwitchingWillBeFinished);
        public void  SetWeaponView(BaseWeaponViewController view);
        public bool  CanFire();
        public bool  CanReload();
        public bool  IsReloading();
        public void  UpdateAmmo(int loaded, int reserved);

        // ── 武器状态机 ────────────────────────────────────
        public void  UpdateWeapon(OnWeaponChangeMode onWeaponChangeMode);
    }
}

// WeaponState 枚举（推测）
enum WeaponState { Idle, Firing, Reloading, Switching, Bolting, Away }
```

```csharp
public abstract class BaseWeaponController : IWeaponController, IEnumerable<BattleWeapon>
```

## 5. 换武器流程

### 5.1 WeaponSwitchingRules — 切换规则配置

`ScriptableObject`，在 Unity 编辑器中配置，控制自动选择逻辑。

```csharp
[CreateAssetMenu(menuName = "Assets/Create/Weapon/WeaponSwitchingRules")]
public class WeaponSwitchingRules : ScriptableObject
{
    // ── 评分常量（决定自动选武器优先级） ──────────────
    private const int SCORE_STATUS_LOADED  = 2000;  // 有子弹优先
    private const int SCORE_STATUS_RESERVE = 1000;  // 有备弹次优
    private const int SCORE_STATUS_MELEE   = 500;   // 近战武器
    private const int SCORE_STATUS_EMPTY   = 0;     // 弹夹空最低

    private const int SCORE_TYPE_PRIMARY   = 2;     // 主武器加分
    private const int SCORE_TYPE_HEAVY     = 1;     // 重型加分

    // ── 切换规则表 ─────────────────────────────────────
    public List<SlotRule> SlotRules;          // 手动切换顺序规则
    //   SlotRule: { FromSlot: 0, ToSlot: 1 }  → 按下切换键时的槽位顺序

    // ── 特殊武器保留配置 ──────────────────────────────
    [Header("Retention Settings")]
    public List<WeaponType> RetentionWeaponTypes;  // 不自动切换的武器类型（如榴弹发射器）

    // ── 方法 ──────────────────────────────────────────
    public int  GetBestWeaponSlot(IBattlePlayer player);  // 选出最优武器（AI死亡自动切换用）
    public int  GetNextSlot(IBattlePlayer player);         // 按规则找下一个切换目标槽
    private int CalculateWeaponScore(BattleWeapon weapon); // 按弹药/类型打分
    private bool IsRetentionWeapon(BattleWeapon weapon);   // 是否保留不切换
    public bool IsSlotRespawnable(byte slot);               // 复活时是否保留该槽
```

### 5.2 完整换武器时序

```
玩家按换武键 / 弹药耗尽自动切换
         │
         ▼
FirstLookBattlePlayer.SwitchWeaponTo(byte targetSlot)
         │
         ▼
IFirstLookWeaponController.SwitchWeapon(targetSlot, onSwitchFinishCallback)
         │
         ├─── 1. 检查 targetSlot 是否有效、是否已是当前槽
         │
         ├─── 2. 记录 PreviousWeapon = CurrentWeapon
         │
         ├─── 3. CurrentWeapon.SwitchToThis(timeWhenFinished)
         │         └── 设置 WeaponState = Switching（启动时序计时）
         │
         ├─── 4. CurrentWeapon.WeaponView.HideWeapon(onHideDone)
         │         └── 播放 Away 动画
         │              └── 等待 HidingDurationMSec 后触发 onHideDone
         │
         ├─── 5. [onHideDone 回调]
         │         ├── CurrentWeapon = _wpnsSlots[targetSlot]
         │         └── CurrentWeapon.WeaponView.ShowWeapon(onSwitchFinishCallback)
         │                  └── 播放 Get 动画
         │                       └── 等待 ShowingDurationMSec 后触发回调
         │
         └─── 6. [onSwitchFinishCallback 回调]
                   └── 触发 OnWeaponSwitchingHasBeenFinished 事件
                        ├── FirstLookBattlePlayer.OnWeaponSwitchingHasBeenFinished()
                        └── 网络同步（通知其他玩家切换了武器槽）
```
